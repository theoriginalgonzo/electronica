# CONTRIBUTING

## Background Knowledge

Assuming:  

- CEG 2350 levels of knowledge in Linux and git.
- An interest in cybersecurity competition participation.

## Standards

AI use is encouraged **with citations**.  Cite any online content (web articles and the like).

## Workflow

1. TEAM CAPTIANS ONLY - [Fork](https://docs.github.com/en/get-started/quickstart/fork-a-repo) this repository
2. Add / invite team members and @mkijowski and @pattonsgirl as contributors to your forked repository. 
3. [Create a branch](https://docs.github.com/en/github/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-and-deleting-branches-within-your-repository) off the main branch for the edits you wish to make
4. Make a [pull request](https://docs.github.com/en/github/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request) in this repository from your branch to the main branch of this repository.
