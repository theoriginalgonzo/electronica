## Course Description

Application of cybersecurity tools is a high-impact, hands-on experience that moves beyond theory into practical application. Students will gain measurable skills in these nine critical areas:

1. Open Source Intelligence
2. Cryptography
3. Password Cracking
4. Log Analysis
5. Network Traffic Analysis
6. Forensics
7. Scanning and Reconaissance
8. Web Application Exploitation
9. Enumeration and Exploitation

## Course Objectives

By the end of this course, students will be able to:

- Analyze complex security artifacts (PCAP files, disk images, and log files) to reconstruct attack timelines.
- Utilize software tools and scripts to accomplish "Capture the Flag" (CTF) challenges.
- Demonstrate technical agility by solving time-constrained "Capture the Flag" (CTF) challenges.
- Collaborate effectively within a team environment to solve high-difficulty security problems.
- Interpret an NCL Scouting Report to communicate personal technical strengths to recruiters.

## Course Expectations

Attendance is mandatory. Students will be graded on team participation, GitHub contributions, performance on exercises in the NCL environment, and individual presentations

## Environment Requirements / Software

NCL Registration: Students will be given a one-time voucher to register for the National Cyber League Spring 2026 Season

Discord: Primary method for out of class communication

Host OS or VM running Linux - Kali preferred. Admin privelges on system. Communicate with instructors immediately if you need access to a system for this course.

GitHub account: students / teams will be using GitHub as a primary means of documenting challenges and progress

Course will require installation and use of tools specific to each cybersecurity knowledge area.

## Grading

Your final grade will be based on class participation, GitHub contribution to the team repository, and performance in the NCL environment.
