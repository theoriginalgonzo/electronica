## 1. Cyberchef local install
- git clone https://github.com/gchq/CyberChef.git
- cd CyberChef
- npm install
- npm start
- go to http://127.0.0.1:8080/ 
## 2. Cyberchef notes
- put the cipher you want to translate into the input box
- find the cipher type you need in the operation search bar
- the output will tell you what the decoded cipher is
## 3. Quipquip notes
- solves letter swap ciphers
- best for long english text
- all you have to do is paste the cipher text and press solve
## 4. Shift gym 
- Open cyberchef and paste iveghny ynxr in the input 
- in the operations panel search ROT13
- output is virtual lake
## 5. French gym 
- open cyberchef and paste Y ln xkv lubj swlzqvkht, A vmzb pjk bbua we ddgs ILQ-GQYU-8026 into input
- in the operations, search vigenere decode and paste qizkwcgqbs into the key box
- the result is: I do not fear computers, I fear the lack of them SKY-QIZK-8026
## 6. Strings gym 
- open cyberchef and drag the image into the input box
- in the operation box find extract strings
- also find regular expression and add SKY to find the sky code
- the code will be highlighted and will be SKY-TVJI-2063
- you can also find it if you use the command line by using the strings command and grep like: strings Steg1.jpg | grep SKY
